
### Curso Git y GitHub

------------

> Proyecto realizado con el objeto de entender y ejecutar el flujo de procesos que involucra un proyecto de **Git** en la plataforma de **GitHub**

|  |  |  |
|--|--|--|
|  | Este repositorio emplea las buenas prácticas aprendidas en la plataforma de **Platzi** a través del [Curso Profesional de Git y GitHub](https://platzi.com/cursos/git-github/) |  |
|  |  |  |

#### Temas vistos
 - [x] Inicialización de repositorios
	 - Locales `git init`
	 - Remotos *(interfaz de GitHub)*
 - [x] Clonación de repositorios
	 - Propios `git clone`
	 - Terceros *(Fork)*
 - [x] Actualización de ramas
	 - Locales `[git fetch + git merge] / [git pull]`
	 - Remotos `git push ...`
 - [x] Mezclas
	 - Git `merge`
	 - GitHub `Pull Request`
 - [ ] *Mucho mas...*
